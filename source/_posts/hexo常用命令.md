**hexo常用命令**

新增文章：

```
hexo n "文件名"
```

新增页面

> hexo n page "页面名称"

启动

```
hexo server
hexo s
```

生成

```
hexo generate 
hexo g
```

部署

```
hexo deploy
hexo d
```

清理缓存

```
hexo clean
```
